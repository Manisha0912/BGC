Background Color

The background-color property sets the background color of an element. The background of an element is the total size of the element, including padding and border (but not the margin)

Text Color

The color property is used to set the color of the text. The color is specified by: a color name - like "red" a HEX value - like "#ff0000" an RGB value - like "rgb(255,0,0)"
